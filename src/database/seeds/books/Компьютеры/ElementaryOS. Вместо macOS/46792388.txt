Об операционной системе elementaryOS. Описывается вид системы, из чего состоит, какими достоинствами и недостатками обладает.




Существует множество операционных систем для ПК. Elementary OS уже несколько лет является одним из самых распространенных настольных дистрибутивов. Сложно превзойти его легкость использования и бесперебойность работы. Elementary OS – дистрибутив Linux специально разработан с простым интерфейсом для предоставления комфортного окружения, в роли которого применяется Pantheon.




Когда и для чего был создан дистрибутив


Проект Elementary начал свою работу 11 лет назад и стартовал, для разработки приложений и тем оформления, но после был сделан полноценный дистрибутив на основе Ubuntu. По сравнению с другими, основная часть приложений разрабатывалась специально для этого проекта. Благодаря этому происходит более качественное взаимодействие с системой, подходящее к её внешнему виду. К тому же, система отличается гораздо более быстрой работой.



Графический интерфейс создавался опытными дизайнерами. Со временем выпустилось много релизов, что позволило операционной системе приобрести большую функциональность. К тому же в наборе программ есть Музыка, Терминал, Файлы, Фото и многое другое.




Из чего состоит Elementary OS


Дистрибутив отличается применением оболочки Pantheon, разработанной именно для этого проекта.


Рабочий стол

Рабочий стол у некоторых вызывает трудности, но здесь сможет разобраться каждый из-за его простоты. В него входит панель и док-бар, где расположены иконки для запуска приложений.
